## Key Concepts

### Exception Handling

Used to handle runtime errors using try, catch and finally blocks.

### LINQ

Language Integrated Query used to filter, search and manipulate collections.

### Delegates

Reference types that can store and invoke methods.

### Async/Await

Used for asynchronous programming to execute tasks without blocking the application flow.

### Extension Methods

Allow adding new methods to existing classes without modifying their source code.
