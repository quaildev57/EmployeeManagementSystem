## Key Concepts

### Controller

Handles incoming HTTP requests and returns responses.

### Routing

Maps URLs to controller actions.

### HTTP Methods

* GET: Retrieve data
* POST: Create data

### Model Binding

Automatically converts incoming JSON into C# objects.

### Dependency Injection

Provides required services automatically without manually creating objects.
